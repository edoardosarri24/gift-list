# gift-list
This project outlines the development of a 'Gift List' application. The application aims to simplify gift-giving by allowing celebrants to create and share personalized wish lists, and guests to claim items, preventing duplicates and enhancing the surprise element.

### Repository Structure
```
.
├── README.md
├── agent.md - Contains the system context and instructions for the AI agent.
├── documentation/ - Contains the project's formal documentation in LaTeX format.
└── todo.md - A file for tracking development tasks and to-do items.
```
